# ♻️ Waste Management System (Java Console Application)

## 📌 Overview

The **Waste Management System** is a Java-based console application designed to manage waste collection processes efficiently.
It supports multiple user roles such as **Admin, Staff, and Customer**, each with specific functionalities.

This project demonstrates the use of **Object-Oriented Programming (OOP)** concepts like classes, enums, encapsulation, and collections.

---

## 🚀 Features

### 👨‍💼 Admin

* Add waste categories
* Delete waste categories
* View all waste collection records

### 👷 Staff

* Register waste collection requests
* Update collection status (Pending / In Progress / Completed)

### 👤 Customer

* Request waste pickup
* View personal collection history

## ⚙️ How to Run

### 🔹 Step 1: Install Requirements

* Install Java JDK (17 or above recommended)
* Install VS Code
* Install **Java Extension Pack** in VS Code

---

### 🔹 Step 2: Compile the Code

Open terminal and run:

```
cd src
javac waste_management_system/Main.java
```

---

### 🔹 Step 3: Run the Program

```
java waste_management_system.Main
```
